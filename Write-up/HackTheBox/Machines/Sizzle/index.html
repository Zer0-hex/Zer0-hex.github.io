    申请域证书

    • 首先本地生成一个证书
    # 创建私钥，需要 4 位以上的密码
openssl genrsa -des3 -out amanda.key 2048
# 用私钥生成一个证书签名请求的 csr 文件
openssl req -new -key amanda.key -out amanda.csr
ls -la amanda.*

    提交 amanda.csr 文件申请证书

    • 下载 Base64 编码的证书
